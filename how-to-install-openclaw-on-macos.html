How to Install OpenClaw on macOS

Complete step-by-step installation guide for MacBook, Mac Mini and iMac

Step 1 — Install Homebrew

Open Terminal and run the following command to install Homebrew (Mac package manager).

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

Step 2 — Install Node.js

OpenClaw requires Node.js. Install the latest Node 22 using Homebrew.

brew install node@22

Step 3 — Install OpenClaw

Run the OpenClaw installation script.

curl -fsSL https://openclaw.ai/install.sh | bash

Step 4 — Onboard OpenClaw

Initialize OpenClaw and install the system daemon.

openclaw onboard --install-daemon

Step 5 — Check Gateway Status

Verify that the OpenClaw gateway is running correctly.

openclaw gateway status

Step 6 — Open the Dashboard

Launch the OpenClaw dashboard in your browser.

openclaw dashboard
