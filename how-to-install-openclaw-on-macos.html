<div class="step">

<h2>Step 1 — Install Homebrew</h2>


<p>Homebrew is required to install dependencies.</p>


<div class="code-box">

<button class="copy-btn" onclick="copyCode(this)">📋</button>

<code>

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

</code>

</div>


</div>


<div class="step">

<h2>Step 2 — Install Node.js</h2>


<p>Install Node.js version 22 using Homebrew.</p>


<div class="code-box">

<button class="copy-btn" onclick="copyCode(this)">📋</button>

<code>

brew install node@22

</code>

</div>


</div>


<div class="step">

<h2>Step 3 — Install OpenClaw</h2>


<p>Run the installation script.</p>


<div class="code-box">

<button class="copy-btn" onclick="copyCode(this)">📋</button>

<code>

curl -fsSL https://openclaw.ai/install.sh | bash

</code>

</div>


</div>


<div class="step">

<h2>Step 4 — Install OpenClaw Daemon</h2>


<div class="code-box">

<button class="copy-btn" onclick="copyCode(this)">📋</button>

<code>

openclaw onboard --install-daemon

</code>

</div>


</div>


<div class="step">

<h2>Step 5 — Check Gateway Status</h2>


<div class="code-box">

<button class="copy-btn" onclick="copyCode(this)">📋</button>

<code>

openclaw gateway status

</code>

</div>


</div>


<div class="step">

<h2>Step 6 — Open the Dashboard</h2>


<div class="code-box">

<button class="copy-btn" onclick="copyCode(this)">📋</button>

<code>

openclaw dashboard

</code>

</div>


</div>
