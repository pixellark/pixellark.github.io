Oceanica International (Pvt) Ltd

Project proposal: Billing + stock management system

Project Overview

After analyzing the current manual system in your factory, we have developed a preliminary plan to digitize it. This will increase efficiency, accuracy, and data analysis capabilities from any device, anywhere. According to this plan, there are three main components:

  • Billing
  • Stock management - to this we have to collect production data, for that we can use tablet device (we can mount it on the wall)
  • sales reporting

Device 1 - PC

This is the main device. Connect the receipt printer to this device as well. All billing functions, stock management, expense recording, and reporting are handled by this device. (If needed, we can extract specific reports from this device.)

Billing Page (Home)

*Bill ID: Bill ID will be automatically generated.

*Customer Name, Vehicle NO: Optional. You can navigate through these input fields using arrow keys.

*Unit Price: This is a pre-determined value that will depend on ice-cube quantity.

Payment Pending: Bills that await payment will be moved to the Pending tab.

*Reprint: You can reprint an old bill using bill ID or the *Recent bills field.

The cursor will focus on the Quantity field by default.

Two bills will be printed when checking out, one for the customer and another for the employees to confirm the order.

**This is not a software requirement specifications (SRS) document. An SRS document shall be provided when the project is started after an advanced payment is delivered, if requested.

Device 2 - Tablet

This is the main device. Connect the receipt printer to this device as well. All billing functions, stock management, expense recording, and reporting are handled by this device. (If needed, we can extract specific reports from this device.)

When taking ice blocks out of the plant, employees can use this device to record it. we could use that data to calculate production.

Before filling the block, it can be input into the system using a PC. The block slots are filled with water and display one color (e.g., blue) while freezing. Once completely frozen, the blocks display another color (e.g., green). The freezing time should be set on the PC. When you click on a green block, the system detects that the cube at that location has been removed.

If it neccesery to detect the employee who takes out the ice cube from a slot, we will have to use a technology like NFC or fingerprint scanner. Cost for such devices and technologies are not listed in the quote.
